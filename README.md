# 🛍️ Flutter E-commerce App

A simple e-commerce app built with Flutter and Provider.

## 🚀 Features
- 📦 Product Listing
- 🔍 Search Functionality
- 🛒 Cart Management
- ❤️ Wishlist Support
- 📲 Responsive UI
